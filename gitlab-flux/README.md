# GitLab-Flux

Install GitOps Kubernetes operator [Flux](https://github.com/weaveworks/flux)

## Prerequisite

- `kubectl` client bin install
- `helm` client bin install
- \*nix shell
- `github` personal token and repo
- `fluxctl`
